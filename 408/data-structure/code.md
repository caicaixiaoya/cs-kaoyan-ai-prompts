## DS-03 快速排序 C 语言实现与逐行解释

**适用场景**：需要理解快速排序的代码实现细节，或想通过看代码加深对 partition 过程的理解时使用。

**提示词**：

> 请用C语言实现快速排序算法，要求：
> 1. 实现标准的 Lomuto 分区方案
> 2. 对每一行代码写注释，解释它在做什么
> 3. 用数组 {5,3,8,1,9,2,4} 手动追踪第一次 partition 的完整过程（逐步展示下标变化）
> 4. 指出代码中容易写错的地方（如边界条件）
> 5. 如果你不确定某个细节，请明确说明

**预期输出示例**：

```c
// 分区函数：将 arr[pivot] 放到正确位置，返回其最终下标
int partition(int arr[], int low, int high) {
    int pivot = arr[high]; // 选最后一个元素为基准
    int i = low - 1;       // i 指向小于pivot区间的末尾
    
    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            swap(&arr[i], &arr[j]); // 把小于pivot的元素移到左侧
        }
    }
    swap(&arr[i+1], &arr[high]); // 基准归位
    return i + 1;
}
```

**使用注意**：

- AI 易错：partition 的边界条件（`j < high` 还是 `j <= high`）常写错，务必手动追踪验证
- 验证方法：把 AI 的代码复制到编译器（如 gcc）运行，用测试数据检验结果
- 追问方向：「请改写为 Hoare 分区方案，并对比两者的区别」
