# 数据结构 提示词库

本目录收录面向 408 数据结构部分的 AI 提示词。

## 分类说明

| 文件 | 用途 |
|------|------|
| [knowledge.md](knowledge.md) | 知识讲解类：让 AI 讲解概念、原理、对比 |
| [quiz.md](quiz.md) | 出题自测类：让 AI 根据章节出题并给出解析 |
| [code.md](code.md) | 代码演练类：让 AI 用 C/C++ 实现算法并逐行解释 |
| [pitfalls.md](pitfalls.md) | 易错点总结类：让 AI 列举常见考研陷阱 |

## 涵盖知识点

- 线性表（顺序表、链表）
- 栈与队列
- 树与二叉树（BST、AVL、B树、B+树、红黑树）
- 图（DFS、BFS、最短路径、最小生成树）
- 查找（顺序、折半、散列）
- 排序（各类排序算法对比）

## 免责声明

> AI 回答存在错误风险，输出结果需结合教材、真题、可信资料自行验证，不构成备考建议。408 考纲以当年官方公布版本为准。
