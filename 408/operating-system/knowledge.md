# 操作系统 · 知识讲解类提示词

> 适用场景：需要 AI 讲解某个知识点的原理、对比相近概念时使用。

---

## OS-01 进程调度算法对比

**适用场景**：需要系统对比各种调度算法的特性，或对某个算法的适用场景有疑惑时使用。

**提示词**：

> 我正在复习考研408操作系统，请对比讲解以下进程调度算法：FCFS、SJF（非抢占）、SRTN（抢占式SJF）、优先级调度、时间片轮转（RR）。
> 要求：
> 1. 用表格对比：是否抢占、适用场景、优点、缺点、考研常考点
> 2. 以以下进程为例，演示FCFS和SJF的调度过程（画甘特图）：
>    P1(到达0,执行4) P2(到达1,执行3) P3(到达2,执行1)
> 3. 指出"饥饿问题"在哪些算法中会出现
> 4. 如果你对某个算法的细节不确定，请明确说明

**预期输出示例**：

```
| 算法 | 抢占 | 优点 | 缺点 | 考研考点 |
|------|------|------|------|---------|
| FCFS | 否 | 公平简单 | 护航效应 | 平均等待时间计算 |
| SJF | 否 | 平均等待最短 | 可能饥饿 | 证明最优性 |
| SRTN | 是 | 响应快 | 实现复杂 | 抢占时机判断 |
...

FCFS甘特图（P1到达0，P2到达1，P3到达2）：
时间: 0  1  2  3  4  5  6  7  8
进程: P1 P1 P1 P1 P2 P2 P2 P3
```

**使用注意**：

- AI 易错：计算平均周转时间时，容易把"等待时间"和"周转时间"混淆（周转时间=完成时间-到达时间）
- 验证方法：手动画甘特图并计算，核查 AI 的结果
- 追问方向：「请用这组进程演示时间片轮转（时间片=2）的调度过程」
