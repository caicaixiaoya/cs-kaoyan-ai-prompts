## OS-02 内存管理出题

**适用场景**：复习完分页/分段/虚拟存储章节，想用 AI 出题检验掌握程度时使用。

**提示词**：

> 我正在备考考研408，请为我出3道关于内存管理的题目，要求：
> 1. 题型：地址变换计算题1道（分页存储，给定页表，求物理地址）、选择题1道（考缺页中断或页面置换）、简答题1道（对比分页与分段的区别）
> 2. 计算题给出逐步解题过程：逻辑地址→页号+页内偏移→查页表→物理地址
> 3. 难度对标近5年真题

**预期输出示例**：

```
【计算题】某系统页面大小为4KB，进程的页表如下：
页号 0 → 帧号 3
页号 1 → 帧号 7
页号 2 → 帧号 1
求逻辑地址 0x1A3C 对应的物理地址。

解：页面大小4KB = 2^12，页内偏移12位
逻辑地址0x1A3C：
页号 = 0x1A3C >> 12 = 0x1 = 1
页内偏移 = 0x1A3C & 0xFFF = 0xA3C
查页表：页号1 → 帧号7
物理地址 = 7 × 4KB + 0xA3C = 0x7000 + 0xA3C = 0x7A3C
```

**使用注意**：

- AI 易错：十六进制地址转换时容易出错，务必手动验算页号和偏移的拆分
- 验证方法：手动将十六进制转二进制，按位数拆分核查
- 追问方向：「请再出一道LRU页面置换算法的题，给出一个页面访问序列，计算缺页次数」
