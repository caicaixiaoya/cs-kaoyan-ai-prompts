# 计算机网络 提示词库

本目录收录面向 408 计算机网络部分的 AI 提示词。

## 分类说明

| 文件 | 用途 |
|------|------|
| [knowledge.md](knowledge.md) | 知识讲解类：让 AI 讲解概念、原理、对比 |
| [quiz.md](quiz.md) | 出题自测类：让 AI 根据章节出题并给出解析 |
| [code.md](code.md) | 代码演练类：让 AI 演示报文分析、子网划分等计算过程 |
| [pitfalls.md](pitfalls.md) | 易错点总结类：让 AI 列举常见考研陷阱 |

## 涵盖知识点

- 计算机网络体系结构（OSI、TCP/IP）
- 物理层
- 数据链路层（滑动窗口、MAC、CSMA/CD）
- 网络层（IP、ARP、ICMP、路由算法）
- 传输层（TCP、UDP）
- 应用层（HTTP、DNS、SMTP 等）
