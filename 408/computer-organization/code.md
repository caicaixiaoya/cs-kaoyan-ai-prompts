## CO-03 补码加减运算溢出判断演示

**适用场景**：需要理解补码加减运算的溢出判断方法，或想通过演示加深理解时使用。

**提示词**：

> 请演示补码加减运算的溢出判断过程，要求：
> 1. 以8位补码为例，分别演示以下运算并判断是否溢出：
>    - (+70) + (+80)
>    - (-70) + (-80)
>    - (+70) - (-80)
> 2. 同时使用两种判断方法：单符号位法 + 双符号位法（变形补码）
> 3. 每步写出二进制过程
> 4. 如果你对溢出判断规则不确定，请明确说明

**预期输出示例**：

```
例：(+70) + (+80)，8位补码

+70 的补码：0 1000110
+80 的补码：0 1010000
           ─────────
结果：      1 0010110  → 符号位为1，但两个正数相加结果为负？→ 溢出！

单符号位法：两个正数相加（符号位均为0），结果符号位为1 → 溢出
双符号位法：+70变形补码 00 1000110，+80变形补码 00 1010000
相加结果：01 0010110 → 双符号位为01（不一致）→ 正溢出
```

**使用注意**：

- AI 易错：双符号位法中"01"表示正溢出、"10"表示负溢出，AI 偶尔会说反
- 验证方法：手动做二进制加法，比较真实结果与8位可表示范围(-128~127)
- 追问方向：「请演示(-70)+(-80)的过程，同样用两种方法判断」
