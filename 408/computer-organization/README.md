# 计算机组成原理 提示词库

本目录收录面向 408 计算机组成原理部分的 AI 提示词。

## 分类说明

| 文件 | 用途 |
|------|------|
| [knowledge.md](knowledge.md) | 知识讲解类：让 AI 讲解概念、原理、对比 |
| [quiz.md](quiz.md) | 出题自测类：让 AI 根据章节出题并给出解析 |
| [code.md](code.md) | 代码演练类：让 AI 演示计算过程（地址计算、流水线等） |
| [pitfalls.md](pitfalls.md) | 易错点总结类：让 AI 列举常见考研陷阱 |

## 涵盖知识点

- 数据的表示与运算（原码/反码/补码、浮点数）
- 存储器层次结构（Cache、主存、虚存）
- 指令系统
- 中央处理器（数据通路、控制器）
- 总线与输入输出系统
