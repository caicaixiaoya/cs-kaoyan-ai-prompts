# 贡献指南

感谢你愿意为本仓库贡献内容！

## 贡献方式

### 提交新 prompt

1. Fork 本仓库
2. 在对应科目目录的对应分类文件中添加新 prompt
3. 严格遵循格式规范（见下方）
4. 提交 Pull Request，标题格式：`feat(科目): 添加xxx提示词`

### 报告问题

发现 prompt 有错误或 AI 输出质量差？请提交 Issue，说明：
- 使用的 AI 工具
- 实际输出内容
- 错误所在（对照教材哪里）

## Prompt 格式规范

每条提示词必须包含以下字段：

```
## [编号] 提示词名称

**适用场景**：一句话说明什么时候用

**提示词**：

> 提示词正文

**预期输出示例**：

（AI 回复的精简示例）

**使用注意**：

- AI 常见错误倾向
- 建议验证方式
- 可选追问方向

---
```

**编号规则：**
- `DS-xx`：数据结构
- `CO-xx`：计算机组成原理
- `OS-xx`：操作系统
- `CN-xx`：计算机网络
- `MI-xx`：面试相关
- `VC-xx`：Vibe Coding 相关

编号从已有最大编号 +1 开始。

## 注意事项

- 不要在仓库中放置版权题目或答案
- 不要暴露任何个人信息（答疑群、他人信息等）
- 提示词语言：全中文
