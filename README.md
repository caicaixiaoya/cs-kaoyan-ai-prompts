# CS-Kaoyan-AI-Prompts

> 面向计算机考研（408 + 复试）的 AI 提示词集合与工作流指南

作者曾备考计算机考研，408 考试期间大量使用 AI 辅助学习。

这是我备考期间用的 deepseek 对话链接，也可以分享给大家：[https://chat.deepseek.com/share/85vbek5aqdarl4udfj](https://chat.deepseek.com/share/85vbek5aqdarl4udfj)

这是网页端链接：[[CS考研 AI提示词平台](https://app7445.acapp.acwing.com.cn/)]([CS考研 AI提示词平台](https://app7445.acapp.acwing.com.cn/))

本仓库的核心理念：**不只给你 prompt，更教你如何验证 AI 的输出，建立自主辨别能力。**

## 适合谁用

| 用户类型 | 核心需求 |
|---------|---------|
| 408 备考考研生 | 用 AI 辅助理解难点、自测、查缺补漏 |
| 复试备考考研生 | 快速搭建可展示的项目、准备面试问答 |
| AI 辅助学习探索者 | 了解如何安全、高效地把 AI 嵌入学习流程 |

---

## 如何使用

1. **选科目**：点击下方对应科目的提示词链接
2. **找场景**：选择 `knowledge`（知识讲解）/ `quiz`（出题自测）/ `code`（代码演练）/ `pitfalls`（易错总结）
3. **复制 prompt**：把提示词粘贴到任意 AI 工具（DeepSeek、Claude、GPT 等）
4. **验证输出**：对照教材或真题确认 AI 的回答，参考每条 prompt 的"使用注意"
